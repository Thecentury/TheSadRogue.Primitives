Struct Distance

Structure representing methods of calculating distance on a grid. You cannot create instances of this class using a constructor -- instead this class contains static instances representing the various distance calculations.

Syntax
[DataContract]
public struct Distance : IEquatable<Distance>, IMatchable<Distance>
Remarks

Provides functions that calculate the distance between two points according to the distance measurement being used. Instances of Distance are also implicitly convertible to both Radius and AdjacencyRule (since both a method of determining adjacent locations and a radius shape are implied by a distance calculation).
